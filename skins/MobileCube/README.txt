MobileCube
Version 3.0.0

Installing:
Unarchive the MobileCube.zip file and place it in the /skins/ directory.


Config requirements:

Skin Folder MUST be called "MobileCube"



$cmail_config['list_cols'] = array('flag', 'from', 'subject', 'date', 'size', 'attachment');

$cmail_config['default_imap_folders'] = array('INBOX', 'Drafts', 'Sent', 'Trash', 'Junk');

$cmail_config['skin_path'] = 'skins/MobileCube/';

$cmail_config['preview_pane'] = TRUE;
